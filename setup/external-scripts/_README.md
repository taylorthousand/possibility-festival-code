# External Scripts

All external CDN scripts and CSS in one place.

## Files

| File | Where to add | Contents |
|------|--------------|----------|
| `head.html` | Site settings → Custom code → Head | GSAP, ScrollTrigger, SplitText, Lenis |

## Load Order (Critical)

1. GSAP Core
2. ScrollTrigger
3. SplitText
4. Lenis CSS + JS

Your custom scripts (text-reveal.js, lenis.js, etc.) go in the **footer** after these.
